# cookiecutters #

Templates for various common project types. To use, run:

```
cookiecutter https://github.com/stubillwhite/cookiecutters --directory python-uv`
```

Project types:

- Python
    - `python-poetry` - Basic Python project with poetry
    - `python-uv` - Basic Python project with uv

- Clojure
    - `clojure-reloaded` - Basic Clojure project using Stuart Sierra's reloaded workflow

- C/C++
    - `cpp` - Basic C++ project with Conan and CTest
